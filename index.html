Projects

Comparative Study of Machine Learning Algorithms on Sentiment Analysis of Product Reviews

Spring 2019
Making machines understand human sentiments would allow us to analyse millions of product reviews that exist on sites like Amazon.
The text reviews were preprocessed and the Bag of Words model was used for feature extraction. We applied various Machine learning algorithms such as Naive Bayes, SVM, Random Forest, etc. and analysed the results.
We created a voted classifier using the mode of the votes of the ML models to obtain a better-performing classifier.
Published "Comparative Study of Machine Learning Algorithms on Sentiment Analysis of Product Reviews" in the Book ICT for Competitive Strategies - Dec 2019

Comparison of Search Algorithms for Autonomous Mobile Rescue Robot

CSCI 5511: Artificial Intelligence I (Fall 2022)
Our goal was to identify effective path-planning algorithms for Rescue robots to detect and make decisions based on various obstacles and topographical perturbations.
We simulated the robot environment for the Rescue Robot using OpenCV and implemented search and sampling-based algorithms for target localization and path planning.
The sampling-based algorithms were able to plan paths much faster due to the lower number of nodes.
